xThemes para Common Utilities
¡por un XOOPS mas bello!

¡Gracias por descargar xThemes!. xThemes es un administrador/framework para temas XOOPS, esto significa que te permite diseñar y desarrollar tus propios temas proporcionándote nuevas herramientas para lograr mejores resultados.

¿Qué me ofrece xThemes que no me ofrezca XOOPS?

La respuesta es muy simple. xThemes te ofrece todas las capacidades de XOOPS y Smarty para la creación de temas, pero además agrega mejoras y nuevas posibilidades a tus temas, haciéndolos fundamentalmente distintos a otros temas. La siguiente es una lista no exhaustiva de lo que xThemes te ofrece:

  • Creador de menús.
    Te permitirá generar fácilmente cualquier tipo de menú e integrarlo en tu tema. Con esta característica olvídate de módulos externos para hacerlo.
  • Integración de configuraciónes.
    Añade a tu tema la capacidad para ser configurado como cualquier módulo XOOPS. Con esta característica podrás ofrecer la posibilidad de ajustar colores, apariencia o lo que tu desees..
  • Plugins Smarty.
    Con xThemes tus temas tendrán un mejor soporte para los plugins de Smarty; sin configuraciones complicadas.
  • Plugins del Tema. Por si fuera poco, xThemes soporta plugins del tema, que son objetos escritos en PHP con los que podrás manipular la información completa que se muestra en tu tema. Además, tus plugins de temas estarán disponibles en cualquier plantilla.

Requerimientos

  • XOOPS 2.5.x (probado en la versión 2.7)
  • Common Utilities 2.2
  • Plugin Advanced Forms

Instalación

Importante: antes de instalar xThemes es necesario tener instalado rmcommon de lo contrario el módulo no podrá instalarse correctamente.

  1. Extrae el contenido del archivo comprimido en tu disco duro.
  2. Sube la carpeta «xthemes» al directorio «modules» de tu instalación XOOPS.
  3. Ingresa al panel de control de XOOPS, luego a la administración de módulos de Common Utilities e instala xThemes como un módulo normal.
  4. ¡Listo! xThemes esta instalado y listo para usarse.

Solución de problemas

Algunas veces xThemes no funcione como se espera inmediatamente después de la instalación. Generalmente esto ocurre debido a un problema con el archivo donde se especifican los módulos activos en el sietema. Para solucionar esto sigue los pasos a continuación:

  1. Localiza la carpeta xoops_data/caches/xoops_cache.
  2. Con un editor de texto o de código abre el archivo xoops_(some alphanumeric chars)_system_modules_active.php.
  3. Podrás observar que existen varias líneas que contienen el nombre de los módulos instalados y activos. Comprueba si existe alguna línea que contenga el nombre de xThemes. Por ejemplo: 
    0 => 'module',
1 => 'module',
2 => 'xthemes',
3 => 'etc...',
  4. Si no está presente el nombre de xthemes, entonces agrega una nueva línea respetando la numeración: 
    4 => 'xthemes',
    Esta nueva línea debe ser agregada entre los paréntesis existentes.

Ejemplo del archivo correcto:

1422009898
return array (
  0 => 'system',
  1 => 'rmcommon',
  2 => 'mywords',
  3 => 'xthemes',
  4 => 'galleries',
  5 => 'vcontrol',
  6 => 'contacts',
  7 => 'contact',
  8 => 'works',
);
